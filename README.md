# E-commerce-website
 Designed interactive web page for buyers to shop and for sellers to list items

![something](https://github.com/HangXiaoCode/requiredPNGs/blob/main/misc/ecommerce%20flowchart.png)

## Major Impact
Designed interactive web page for buyers to shop and for sellers to list items using **JavaScript**, and **RESTful APIs**.

Constructed the item listing and searching based on **Spring MVC** framework.

Implemented **JDBC authentication** provided by Spring Security to ensure security workflow.

Deployed relational database (**MySQL**) operations with Hibernate for simpler queries.
